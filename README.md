## 命名規則の確立
### ディレクトリ
- １単語で収める．２単語以上にはしない．
- ex: domain, infrastructure
### 変数
- Lower Camel Case
- ex: userId, accessToken
### 関数
- Lower Camel Case
- ex: getUserId, getUserName
### クラス
- Upper Camel Case
- ex: UserService, LoginRepository
